= Configuration Reference
:toc:

== Runtime Configuration

The following configuration properties are available at runtime:

[cols="3,1,1,3"]
|===
| Property | Type | Default | Description

| `quarkus.agent.base-url`
| `string`
| `http://localhost:11434`
| Base URL of the OpenAI-compatible LLM API. Supports Ollama, OpenAI, vLLM, and any compatible endpoint.
|===

== Annotation Attributes

=== @Agent

[cols="2,1,1,3"]
|===
| Attribute | Type | Default | Description

| `provider`
| `String`
| `"ollama"`
| LLM provider identifier

| `model`
| `String`
| _(required)_
| Model name (e.g., `llama3`, `gpt-4`, `tinyllama`)

| `architecture`
| `Architecture`
| _(required)_
| Agent architecture enum value

| `maxIterations`
| `int`
| `10`
| Max iterations for iterative architectures

| `temperature`
| `double`
| `0.7`
| LLM sampling temperature

| `systemPrompt`
| `String`
| `""`
| System prompt prepended to LLM calls

| `quality`
| `InferenceQuality`
| `MEDIUM`
| Inference quality tier (HIGH, MEDIUM, LOW) for routing
|===

=== @CrewMember

[cols="2,1,1,3"]
|===
| Attribute | Type | Default | Description

| `role`
| `String`
| _(required)_
| Agent role name

| `backstory`
| `String`
| `""`
| Agent background description

| `task`
| `String`
| _(required)_
| Task assignment for this agent
|===

=== @PipelineStep

[cols="2,1,1,3"]
|===
| Attribute | Type | Default | Description

| `name`
| `String`
| _(required)_
| Step identifier

| `systemPrompt`
| `String`
| _(required)_
| System prompt for this processing step

| `order`
| `int`
| `0`
| Execution order (ascending)
|===

=== @DebateAgent

[cols="2,1,1,3"]
|===
| Attribute | Type | Default | Description

| `name`
| `String`
| _(required)_
| Debater name

| `personality`
| `String`
| `""`
| Personality description for the debater
|===

=== @Generate

Method-level. Marks the method that generates the initial prompt. Receives `@UserPrompt` parameter.

=== @EvaluateOutput

Method-level. Marks the method that evaluates output quality. Must return `boolean`.

=== @UserPrompt

Parameter-level. Marks the parameter that receives the user's input prompt.

== InferenceQuality Enum

[cols="1,3"]
|===
| Value | Description

| `HIGH` | Most capable model (Claude Opus, GPT-4o)
| `MEDIUM` | Balanced cost/quality (default)
| `LOW` | Fast and cheap (Groq, tinyllama)
|===
